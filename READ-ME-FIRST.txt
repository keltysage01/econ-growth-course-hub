────────────────────────────────────────────────
1) TO SEE THE SITE (no tools to install)
────────────────────────────────────────────────
• On a Mac: double-click  START-PREVIEW.command
  → it opens the site in your browser at http://localhost:8123
  (If macOS warns "unidentified developer": right-click it → Open → Open.)

• On Windows / no terminal: install the free "Live Server" extension in
  VS Code, open this folder, right-click index.html → "Open with Live Server".

Important: just double-clicking index.html directly WON'T load the styles
(the images/fonts use absolute paths). Use the preview above instead.
